# üôè Contributing to the CL Extension

We welcome everyone to contribute to the **CL** extension! We appreciate any contribution, whether it be to documentation or code. 

For ideas on where to help out, check out the [open issues](https://github.com/IBM/vscode-clle/issues) and espically those labeled as [good first issue](https://github.com/IBM/vscode-clle/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22good%20first%20issue%22). Once you are happy to share your changes, please create a pull request and it will be reviewed by the maintainers of this project.

## Getting Started

1. Install [VS Code](https://code.visualstudio.com/download) and [Node.js](https://nodejs.org/en/download/package-manager).
2. Create a [fork](https://github.com/IBM/vscode-clle/fork) of this repository.
3. Clone your fork.
   ```sh
   git clone https://github.com/your-username/vscode-clle.git
   cd vscode-clle
   ```
4. Install all extension dependencies.
    ```sh
    npm install
    ```
5. Use `Launch Client` from VS Code's `Run and Debug` view.
6. Use `Attach to Server` from VS Code's `Run and Debug` view if you also want to debug the language server.
